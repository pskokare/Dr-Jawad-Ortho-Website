# Dr. Germantan Website

A professional and modern healthcare website built with HTML, Tailwind CSS, and JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Navigation**: Sticky navbar with mobile menu support
- **Contact Form**: Functional contact form with validation
- **Smooth Scrolling**: Seamless navigation between sections
- **Accessibility**: Keyboard navigation and semantic HTML
- **Performance**: Optimized for fast loading and smooth interactions

## Technologies Used

- **HTML5**: Semantic markup for better accessibility
- **Tailwind CSS**: Utility-first CSS framework for modern styling
- **Vanilla JavaScript**: Interactive features and animations
- **Font Awesome**: Professional icons for UI elements

## File Structure

```
Dr.GERMANTAN_WEBSITE/
├── index.html          # Main HTML file
├── styles.css          # Custom CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # Project documentation
```

## Sections

1. **Navbar**: Sticky navigation with mobile responsiveness
2. **Hero Section**: Eye-catching introduction with call-to-action buttons
3. **About Section**: Professional information about Dr. Germantan
4. **Services Section**: Overview of medical services offered
5. **Contact Section**: Contact information and contact form
6. **Footer**: Additional links and social media connections

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website will load with all functionality working immediately

## Customization

### Colors
The primary color is blue (`#3B82F6`). You can modify this in the HTML classes or add custom CSS variables.

### Content
Update the content directly in the HTML file:
- Change text in headings and paragraphs
- Update contact information
- Modify service descriptions
- Add your own images by replacing placeholder URLs

### Styling
Additional custom styles can be added to `styles.css`:
- Custom animations
- Additional hover effects
- Responsive adjustments
- Theme variations
